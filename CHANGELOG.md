# Change Log

## [0.3.2] - 20.11.2025

- Changed how protocols are parsed before being visualised. Fields of protocol definitions may consist of expressions such as identifiers and function calls --- not only string literals.
- Changed well-formedness check to use a revised definition of well-formedness.

## [0.3.1] - 01.01.2025

- Fixed bug that removed initial node if label changed

## [0.3.0] - 01.01.2025

- Added possibility to read and edit subscriptions
- Added possibility to change log type
- Added well-formedness check before save
- Added possibility to download diagram as PNG
- Updated state design

## [0.2.0] - 29.11.2024

- Added the possibility of movable edges. Create handles and drag edges, to create a simpler overview of the diagram.

## [0.1.0] - 19.11.2024

- Initial release
